# OceanView Resort - Hotel Management System

A full-stack **Hotel and Resort Management Web System** built with Java and modern web technologies. Features room management, guest registration, reservations, billing, and an admin dashboard with role-based access control.

## Key Features

- **Room Management** — Add, edit, and manage room inventory with pricing
- **Guest Management** — Register guests and manage contact information
- **Reservation System** — Book rooms with check-in/check-out management
- **Billing & Checkout** — Generate bills and process guest checkouts
- **Admin Dashboard** — Real-time statistics and activity monitoring
- **Role-Based Access** — Admin and staff permission levels
- **System Health Checks** — Built-in diagnostic tools
- **Database Management** — MySQL schema with repair scripts

## Technologies & Tech Stack

| Technology | Purpose |
|---|---|
| Java 17 | Backend application logic |
| Maven | Build automation |
| Apache Tomcat | Web application server |
| MySQL (XAMPP) | Relational database |
| HTML/CSS/JS | Frontend user interface |
| RBAC | Role-based access control |

## Setup and Installation

1. **Prerequisites:** Java 17+, XAMPP (Apache, MySQL, Tomcat), Maven
2. Clone the repository:
   ```
   git clone https://github.com/avishka-d-rajapaksha/OceanViewResort.git
   ```
3. Start XAMPP services (Apache, MySQL, Tomcat)
4. Initialize the database:
   ```
   mysql -u root < resort.sql
   ```
5. Deploy: `start_resort_system.bat`
6. Access at `http://localhost:8080/OceanViewResort`
7. Login: `admin` / `admin`
